# TrueConf

Тестовое задание было сделано с использованием Microsoft Visual Studio.

Представляет из себя 5 функций:
	void fillVec(std::vector<int> *, int) - заполняет переданный вектор некоторым количеством значений.

	void fillMap(std::map<int, int> *, int) - заполняет переданный словарь некоторым количеством значений.

	void deleteFromVec(std::vector<int> *) - удаляет из переданного вектора случайное количество элементов с конца.

	void deleteFromMap(std::map<int, int> *) - удаляет из переданного словаря случайное количество элементов с конца.

	void synchronization(std::vector<int> *, std::map<int, int> *) - синхронизирует переданный вектор и словарь
